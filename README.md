# Web Portfolio

This is a web portfolio project showcasing my skills in JavaScript, HTML, and CSS.

## Project Structure

The project has the following files:

- `src/css/styles.css`: This file is where you can define the styles for your web page. You can add CSS rules to customize the appearance of your portfolio.

- `src/js/script.js`: This file is where you can write your JavaScript code. You can add functions and logic to enhance the interactivity of your web page.

- `src/index.html`: This file is the main HTML file of your web page. You can structure the content of your portfolio using HTML tags and link the CSS and JavaScript files.

## Getting Started

To view the initial structure of your web page, open the `index.html` file in a web browser. From there, you can edit the CSS and JavaScript files to add your own styles and functionality.
