# Web Portfolio

This is a web portfolio project showcasing my skills in JavaScript, HTML, and CSS. The portfolio is designed to highlight my projects, experience, and technical expertise in web development.

## Project Structure

The project has the following files:

- `src/css/styles.css`: This file contains the styles for the web page. It includes CSS rules to customize the appearance of the portfolio.
- `src/js/script.js`: This file contains JavaScript code to add interactivity and dynamic functionality to the web page.
- `src/index.html`: This is the main HTML file of the web page. It structures the content of the portfolio and links the CSS and JavaScript files.

## Features

- Responsive design for optimal viewing on various devices.
- Interactive elements powered by JavaScript.
- Clean and modern UI/UX design.

## Getting Started

To view the portfolio:

1. Clone the repository to your local machine.
2. Open the `src/index.html` file in a web browser.
3. Customize the CSS and JavaScript files to add your own styles and functionality.

## Technologies Used

- **HTML5**: For structuring the content of the web page.
- **CSS3**: For styling and layout.
- **JavaScript**: For adding interactivity and dynamic behavior.
